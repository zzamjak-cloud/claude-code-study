## 시작하기

### 1. 애플리케이션 실행

Game Planner를 실행하면 초기 설정 화면이 표시됩니다.

### 2. API Key 설정

Game Planner는 Google Gemini API를 사용하므로 API Key가 필요합니다.

1. 화면 상단의 **설정 아이콘**을 클릭합니다.
2. **Gemini API Key** 필드에 API Key를 입력합니다.
3. (선택 사항) **Notion API Key** 및 **Database ID**를 입력하면 Notion 연동이 가능합니다.
4. **저장** 버튼을 클릭합니다.

**API Key 발급 방법:**
- Gemini API Key: [Google AI Studio](https://aistudio.google.com/app/apikey)에서 발급
- Notion API Key: [Notion Integrations](https://www.notion.so/my-integrations)에서 발급

### 3. 화면 구성

Game Planner는 크게 3개의 영역으로 구성됩니다:

- **왼쪽 사이드바**: 세션 목록 및 관리
- **가운데 채팅 패널**: AI와 대화하며 작업 진행
- **오른쪽 프리뷰 패널**: 생성된 문서 미리보기 및 관리

## 기획서 작성하기

### 새 기획 세션 만들기

1. 왼쪽 사이드바 상단의 **기획** 탭을 선택합니다.
2. **+ 새 기획 세션** 버튼을 클릭합니다.
3. 세션 제목을 입력합니다 (예: "RPG 게임 전투 시스템 기획").

### 템플릿 선택

새 세션을 만들면 템플릿 선택 화면이 표시됩니다.

1. **기획 템플릿** 목록에서 사용할 템플릿을 선택합니다.
   - **기본 게임 기획서 템플릿**: 게임 개요, 핵심 메커니즘, 시스템 설계 등 포괄적인 구조
   - **기본 스토리 템플릿**: 스토리 중심 게임을 위한 구조
   - 또는 직접 만든 커스텀 템플릿 사용
2. **시작하기** 버튼을 클릭합니다.

### AI와 대화하며 기획서 작성

1. 채팅 입력창에 요청사항을 입력합니다.
   - 예: "모바일 RPG 게임의 전투 시스템을 기획해줘"
   - 예: "장비 강화 시스템을 추가해줘"
2. AI가 기획서를 작성하며, 오른쪽 프리뷰 패널에서 실시간으로 확인할 수 있습니다.
3. 수정이 필요한 경우, 구체적으로 요청합니다.
   - 예: "전투 시스템에 스킬 쿨타임 개념을 추가해줘"
   - 예: "장비 등급을 5단계로 수정해줘"

### 참조 파일 활용

기존 문서나 데이터를 참고하여 기획서를 작성할 수 있습니다.

1. 오른쪽 프리뷰 패널 상단의 **레퍼런스** 탭을 클릭합니다.
2. **참조 파일 추가** 버튼을 클릭하거나, 파일을 드래그 앤 드롭합니다.
3. 지원 파일 형식:
   - **PDF**: 기획서, 보고서 등
   - **Excel/CSV**: 게임 밸런스 데이터, 아이템 테이블 등
   - **Markdown/Text**: 기존 기획 문서
4. AI는 대화 중 관련 있는 참조 파일을 자동으로 활용합니다.

**팁:**
- 참조 파일이 많을 경우, AI가 자동으로 관련성 높은 파일만 선택하여 사용합니다.
- 참조 파일 요약 기능을 활용하면 API 비용을 절감할 수 있습니다.

## 게임 분석하기

### 새 분석 세션 만들기

1. 왼쪽 사이드바 상단의 **분석** 탭을 선택합니다.
2. **+ 새 분석 세션** 버튼을 클릭합니다.
3. 분석할 게임명을 입력합니다 (예: "Clash Royale").

### 템플릿 선택

1. **분석 템플릿** 목록에서 사용할 템플릿을 선택합니다.
   - **기본 게임 분석 템플릿**: 게임 개요, 핵심 메커니즘, 수익화 전략 등
   - 또는 직접 만든 커스텀 분석 템플릿 사용
2. **분석 시작** 버튼을 클릭합니다.

### AI 자동 분석

AI가 다음과 같은 작업을 자동으로 수행합니다:

1. **Google Search Grounding**: 최신 게임 정보를 실시간으로 검색합니다.
2. **분석 보고서 생성**: 템플릿 구조에 따라 체계적인 분석 보고서를 작성합니다.
3. **실시간 업데이트**: 분석 진행 상황을 오른쪽 프리뷰 패널에서 확인할 수 있습니다.

### 추가 질문 및 심화 분석

분석이 완료된 후에도 추가 질문을 할 수 있습니다.

- 예: "이 게임의 초반 유저 리텐션 전략을 더 자세히 분석해줘"
- 예: "경쟁 게임들과 비교 분석을 추가해줘"

**주의사항:**
- 게임 분석은 Google Search를 활용하므로 인터넷 연결이 필요합니다.
- 분석 결과는 실시간 검색 결과를 기반으로 하므로, 최신 정보를 반영합니다.

## 템플릿 관리

템플릿은 AI가 기획서나 분석 보고서를 작성할 때 사용하는 구조와 지침을 정의합니다.

### 템플릿 관리 화면 열기

1. 왼쪽 사이드바 하단의 **템플릿 관리** 버튼을 클릭합니다.
2. 템플릿 목록이 표시됩니다 (기획 템플릿 / 분석 템플릿).

### 새 템플릿 만들기

1. **+ 새 템플릿** 버튼을 클릭합니다.
2. 템플릿 유형을 선택합니다 (기획 / 분석).
3. 템플릿 이름과 설명을 입력합니다.
4. **편집** 버튼을 클릭하여 템플릿 내용을 작성합니다.

### 템플릿 편집기 사용법

템플릿 편집기는 리치 텍스트 에디터(Tiptap)를 기반으로 합니다.

**기본 기능:**
- **서식 적용**: 굵게, 기울임, 밑줄, 취소선
- **헤더**: H1 ~ H3 지원
- **리스트**: 순서 있는 리스트, 순서 없는 리스트
- **인용**: 블록 인용
- **코드**: 인라인 코드, 코드 블록
- **링크**: URL 링크 삽입

**이모지 추가:**
- **방법 1**: 오른쪽 상단의 **이모지** 버튼 클릭 후 선택
- **방법 2**: 에디터에서 `:` 입력 후 이모지 검색 (예: `:smile`)

**줌 기능:**
- 오른쪽 상단의 **+** / **-** 버튼으로 에디터 크기 조절 (50% ~ 200%)

**템플릿 작성 팁:**
- 명확한 섹션 구조를 만들어 AI가 일관된 문서를 생성하도록 합니다.
- 각 섹션에 상세한 설명과 예시를 포함하면 더 좋은 결과를 얻을 수 있습니다.
- 기획 템플릿: 게임의 핵심 요소, 시스템, 밸런스 등을 포함
- 분석 템플릿: 게임 개요, 메커니즘, 수익화, 시장 분석 등을 포함

### 템플릿 순서 변경

1. 템플릿 목록에서 **≡** 아이콘을 드래그하여 순서를 변경합니다.
2. 순서는 자동으로 저장됩니다.

### 템플릿 삭제

1. 삭제할 템플릿의 **삭제** 버튼을 클릭합니다.
2. 확인 다이얼로그에서 **삭제** 버튼을 클릭합니다.

**주의:**
- 기본 템플릿은 삭제할 수 없습니다.
- 템플릿을 삭제해도 해당 템플릿으로 만든 세션은 삭제되지 않습니다.

## 참조 파일 관리

참조 파일은 기획서 작성 시 AI가 참고할 수 있는 문서나 데이터입니다.

### 참조 파일 추가

**방법 1: 파일 선택**
1. 오른쪽 프리뷰 패널의 **레퍼런스** 탭을 클릭합니다.
2. **참조 파일 추가** 버튼을 클릭합니다.
3. 파일 탐색기에서 파일을 선택합니다.

**방법 2: 드래그 앤 드롭**
1. 파일 탐색기에서 파일을 드래그합니다.
2. Game Planner 창으로 드롭합니다.

**지원 파일 형식:**
- **PDF**: `.pdf`
- **Excel**: `.xlsx`, `.xls`
- **CSV**: `.csv`
- **Markdown**: `.md`
- **텍스트**: `.txt`
- **Google Spreadsheet**: URL 지원 (공개 링크만 가능)

### 참조 파일 관리

**파일 요약 보기:**
- 각 파일 카드에서 **요약 보기** 버튼을 클릭하면 AI가 생성한 요약을 확인할 수 있습니다.
- 요약은 비용 최적화를 위해 자동으로 생성됩니다.

**파일 삭제:**
1. 삭제할 파일의 **삭제** 버튼을 클릭합니다.
2. 확인 다이얼로그에서 **삭제** 버튼을 클릭합니다.

### 참조 파일 활용 방식

AI는 대화 중 다음과 같이 참조 파일을 활용합니다:

1. **자동 관련성 필터링**: 사용자 메시지와 관련성 높은 파일만 선택
2. **스마트 포함 전략**:
   - **작은 파일** (< 5,000자): 전체 내용 사용
   - **중간 파일** (5,000 ~ 100,000자): 요약 + 일부 내용 사용
   - **큰 파일** (> 100,000자): 요약만 사용
3. **최대 3개 파일**: 관련성 높은 순으로 최대 3개까지 포함

**비용 최적화 팁:**
- 참조 파일이 많을 경우, 파일명과 요약을 명확하게 작성하면 AI가 더 정확하게 관련 파일을 선택합니다.
- 불필요한 파일은 삭제하여 API 비용을 절감할 수 있습니다.

## 버전 관리

문서의 버전을 저장하고 복원할 수 있습니다.

### 버전 저장

1. 오른쪽 프리뷰 패널의 **버전** 탭을 클릭합니다.
2. **새 버전 저장** 버튼을 클릭합니다.
3. 버전 설명을 입력합니다 (예: "전투 시스템 초안", "스킬 밸런스 조정").
4. **저장** 버튼을 클릭합니다.

### 버전 복원

1. 버전 목록에서 복원할 버전을 선택합니다.
2. **복원** 버튼을 클릭합니다.
3. 확인 다이얼로그에서 **복원** 버튼을 클릭합니다.
4. 문서가 선택한 버전으로 되돌아갑니다.

### 버전 비교

1. 버전 목록에서 비교할 버전을 선택합니다.
2. **비교** 버튼을 클릭합니다.
3. 현재 문서와 선택한 버전의 차이점이 표시됩니다.
   - **빨간색**: 삭제된 내용
   - **초록색**: 추가된 내용

**활용 사례:**
- 중요한 변경 전에 버전 저장
- 여러 아이디어를 시도하고 비교
- 이전 버전으로 되돌리기

## 문서 검증

AI를 활용하여 작성한 문서를 검증할 수 있습니다.

### 검증 실행

1. 오른쪽 프리뷰 패널의 **검증** 탭을 클릭합니다.
2. **문서 검증 실행** 버튼을 클릭합니다.
3. AI가 문서를 분석하여 체크리스트를 생성합니다.

### 체크리스트 관리

검증 결과는 다음과 같은 카테고리로 구성됩니다:

- **완성도**: 문서의 전반적인 완성도
- **일관성**: 용어, 수치, 로직의 일관성
- **구체성**: 설명의 구체성 및 명확성
- **실현 가능성**: 개발 가능성 및 현실성

**체크리스트 항목:**
- ✅ **통과**: 문제 없음
- ⚠️ **경고**: 개선 필요
- ❌ **실패**: 심각한 문제

**수동 체크:**
- 각 항목을 클릭하여 수동으로 체크 상태를 변경할 수 있습니다.
- 자신만의 체크리스트를 추가할 수도 있습니다.

### 검증 결과 활용

1. 문제가 발견된 항목을 확인합니다.
2. 채팅 패널에서 AI에게 수정을 요청합니다.
   - 예: "검증 결과에서 지적된 전투 밸런스 문제를 수정해줘"
3. 수정 후 다시 검증을 실행하여 확인합니다.

## Notion 연동

작성한 기획서나 분석 보고서를 Notion으로 자동 저장할 수 있습니다.

### Notion 설정

1. 화면 상단의 **설정 아이콘**을 클릭합니다.
2. **Notion API Key** 및 **Database ID**를 입력합니다.
3. **저장** 버튼을 클릭합니다.

**Notion Database 준비:**
- Notion에서 데이터베이스를 만들고, Integration을 연결합니다.
- Database ID는 Notion 페이지 URL에서 확인할 수 있습니다.
  - 예: `https://notion.so/workspace/{DATABASE_ID}?v=...`

### 문서 저장

1. 오른쪽 프리뷰 패널에서 **Notion에 저장** 버튼을 클릭합니다.
2. AI가 자동으로 마크다운을 Notion 블록으로 변환합니다.
3. 저장이 완료되면 Notion 페이지 URL이 표시됩니다.

**지원 형식:**
- 헤더 (H1, H2)
- 텍스트 및 서식 (굵게, 기울임, 밑줄, 취소선, 코드)
- 리스트 (순서 있는 리스트, 순서 없는 리스트, 2단계 중첩)
- 인용
- 코드 블록
- 구분선

**주의사항:**
- H3 이하의 헤더는 H2로 변환됩니다 (Notion 제약).
- 블록당 최대 2,000자 제한이 있습니다.
- 복잡한 테이블이나 이미지는 지원하지 않을 수 있습니다.

## 세션 관리

### 세션 전환

왼쪽 사이드바에서 세션을 클릭하면 해당 세션으로 전환됩니다.

### 세션 내보내기 및 불러오기

**내보내기:**
1. 내보낼 세션을 우클릭하거나, 세션 액션 메뉴를 엽니다.
2. **내보내기** 버튼을 클릭합니다.
3. 저장 위치를 선택합니다.
4. JSON 파일로 저장됩니다.

**불러오기:**
1. 왼쪽 사이드바 하단의 **세션 불러오기** 버튼을 클릭합니다.
2. 이전에 내보낸 JSON 파일을 선택합니다.
3. 세션이 목록에 추가됩니다.

**활용 사례:**
- 세션을 백업하여 안전하게 보관
- 다른 컴퓨터로 세션 이동
- 팀원과 세션 공유

### 세션 삭제

1. 삭제할 세션을 우클릭하거나, 세션 액션 메뉴를 엽니다.
2. **삭제** 버튼을 클릭합니다.
3. 확인 다이얼로그에서 **삭제** 버튼을 클릭합니다.

**주의:**
- 삭제된 세션은 복구할 수 없습니다.
- 중요한 세션은 삭제 전에 내보내기를 권장합니다.

## FAQ 및 문제 해결

### Q1. API Key를 잘못 입력했어요. 어떻게 수정하나요?

**A:** 화면 상단의 설정 아이콘을 클릭하여 API Key를 다시 입력하고 저장하세요.

### Q2. AI 응답이 멈춘 것 같아요.

**A:** 다음을 시도해보세요:
1. 네트워크 연결 확인
2. API Key가 유효한지 확인 (할당량 초과 여부)
3. 애플리케이션 재시작

### Q3. 참조 파일이 너무 커서 추가되지 않아요.

**A:** 파일 크기는 최대 100,000자로 제한됩니다. 파일을 분할하거나, 필요한 부분만 추출하여 추가하세요.

### Q4. Notion 저장이 실패했어요.

**A:** 다음을 확인하세요:
1. Notion API Key가 유효한지 확인
2. Database ID가 정확한지 확인
3. Notion Integration이 해당 데이터베이스에 연결되어 있는지 확인
4. 네트워크 연결 확인

### Q5. 버전을 복원했는데 최신 내용이 사라졌어요.

**A:** 버전 복원은 문서를 선택한 버전으로 되돌립니다. 복원 전에 현재 버전을 저장하는 것을 권장합니다.

### Q6. 템플릿을 수정했는데 이미 만든 세션에는 반영되지 않아요.

**A:** 템플릿은 새 세션을 만들 때만 적용됩니다. 기존 세션에는 영향을 주지 않습니다.

### Q7. 게임 분석이 너무 오래 걸려요.

**A:** 게임 분석은 Google Search를 활용하여 최신 정보를 수집하므로 시간이 걸릴 수 있습니다. 인터넷 속도와 검색 결과량에 따라 달라질 수 있습니다.

### Q8. 기획서나 분석 보고서가 중간에 잘렸어요.

**A:** 긴 문서를 작성할 때 토큰 제한으로 인해 잘릴 수 있습니다. 해결 방법:
1. 채팅창에 "⚠️ 경고: 내용이 잘렸을 수 있습니다"라는 메시지가 표시되는지 확인
2. "계속 작성해줘" 또는 "9번 항목을 완성해줘"라고 요청
3. 개발자 콘솔(F12)에서 "[API] 최대 토큰 수에 도달" 메시지 확인
4. 더 간결한 템플릿 사용을 고려

### Q9. 이모지를 빠르게 입력하고 싶어요.

**A:** 템플릿 에디터에서 `:` 를 입력하면 이모지 검색 팝업이 표시됩니다. 원하는 이모지를 검색하여 빠르게 삽입할 수 있습니다.

### Q10. API 비용이 걱정돼요. 어떻게 절감할 수 있나요?

**A:** 다음 방법을 사용하세요:
1. 참조 파일 요약 기능 활용
2. 불필요한 참조 파일 삭제
3. 짧고 명확한 요청 사용
4. 반복적인 수정보다 한 번에 명확하게 요청

### Q11. 데이터가 저장되는 위치가 궁금해요.

**A:** Game Planner는 로컬 데스크톱 애플리케이션으로, 모든 데이터는 컴퓨터에 저장됩니다. 클라우드에 저장되지 않으므로 개인정보 보호가 보장됩니다.
